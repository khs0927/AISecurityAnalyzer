Android 인텐트 스킴

Android에서는 인텐트 스킴을 사용하여 앱으로 직접 연결할 수 있습니다:

인텐트로 Expo Go 열기

참고: 인텐트 스킴은 Android 기기에서만 작동하며, 모든 브라우저에서 지원되지 않을 수 있습니다. Chrome 브라우저를 사용하는 것이 가장 좋습니다.

인텐트 스킴이란? Android에서 앱을 호출하는 특수한 형식의 URL입니다. 이 방식은 앱이 설치되어 있지 않은 경우에도 Play Store로 리디렉션할 수 있습니다.

인텐트 URL 형식

intent://exp.host/@hsml/heart-care-expo?runtime-version=exposdk:50.0.0#Intent;scheme=exp;package=host.exp.exponent;end

대체 연결 방법

exp:// 스킴으로 시도 expo:// 스킴으로 시도 웹 리디렉션으로 시도

안내: 위 버튼들이 작동하지 않으면 Expo Go 앱에서 다음 URL을 직접 입력해보세요:

exp://exp.host/@hsml/heart-care-expo?runtime-version=exposdk:50.0.0

Expo Go 앱 → 프로필 탭 → URL 입력 → 위 URL 붙여넣기

문제 해결

  • Expo Go 앱이 최신 버전인지 확인하세요
  • Chrome 브라우저를 사용하세요
  • 인터넷 연결이 안정적인지 확인하세요
  • 다른 버튼과 링크를 시도해보세요
  • 인텐트 스킴이 작동하지 않으면 Expo Go 앱을 먼저 실행해보세요

iOS 유니버설 링크

iOS에서는 Universal Links와 커스텀 URL 스킴을 사용할 수 있습니다:

exps:// 스킴으로 열기 (권장) 유니버설 링크 시도

참고: iOS에서는 Safari 브라우저를 사용하는 것이 가장 좋습니다. Safari에서 "방문 웹사이트에서 앱 열기"를 허용해야 합니다.

유니버설 링크란? 웹 URL과 앱을 연결하는 Apple의 기술입니다. 앱이 설치되어 있으면 앱을 열고, 그렇지 않으면 웹 페이지로 리디렉션합니다.

대체 연결 방법

exp:// 스킴으로 시도 expo:// 스킴으로 시도 웹 리디렉션으로 시도

안내: 위 버튼들이 작동하지 않으면 Expo Go 앱에서 다음 URL을 직접 입력해보세요:

exp://exp.host/@hsml/heart-care-expo?runtime-version=exposdk:50.0.0

Expo Go 앱 → 프로필 탭 → URL 입력 → 위 URL 붙여넣기

문제 해결

  • Safari 브라우저를 사용하세요
  • iOS 설정 앱에서 Safari 설정을 확인하세요
  • Expo Go 앱이 최신 버전인지 확인하세요
  • 다른 버튼과 링크를 시도해보세요
  • 링크를 길게 눌러 "Safari에서 열기" 옵션을 선택해보세요
