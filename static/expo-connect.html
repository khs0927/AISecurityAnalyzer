문제 해결

Expo 앱 연결에 문제가 있을 경우:

  • Expo Go 앱을 최신 버전으로 업데이트하세요
  • 앱을 완전히 종료하고 다시 시작해보세요
  • 네트워크 연결이 안정적인지 확인하세요
  • Expo Go 앱에서 로그인 후 시도해보세요
  • URL에 runtime-version=exposdk:50.0.0 파라미터가 포함되어 있는지 확인하세요
  • "The request for this experience was malformed" 오류가 나타나면 SDK 버전 불일치 문제입니다

기존 프로젝트 사용
