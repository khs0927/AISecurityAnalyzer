HeartCare Expo 스킴 연결

모든 Expo 스킴 방식 시도

아래 링크들은 다양한 Expo 스킴 방식을 사용하여 HeartCare 앱을 열기를 시도합니다.

방법 1: exp:// 스킴 사용 방법 2: exps:// 스킴 사용 방법 3: expo:// 스킴 사용 방법 4: u.expo.dev 사용
참고: 위 링크 중 하나를 클릭하면 Expo Go 앱이 자동으로 실행되고 HeartCare 앱이 로드됩니다. 여러 방법을 시도해보세요.

직접 URL 복사하기

Expo Go 앱을 열고 다음 URL을 직접 입력하세요:

exp://exp.host/@hsml/heart-care-expo?runtime-version=exposdk:50.0.0

연결이 안 되나요?

다음 사항을 확인하세요:

  • Expo Go 앱을 최신 버전으로 업데이트하세요
  • 앱이 설치되어 있지 않으면 App Store에서 "Expo Go"를 설치하세요
  • Safari 대신 Chrome 브라우저로 시도해보세요
  • 앱을 완전히 종료하고 다시 시도해보세요
  • 다른 네트워크를 사용해보세요 (예: WiFi에서 셀룰러 데이터로 전환)

계속 문제가 있다면 홈으로 돌아가기를 클릭하여 다른 연결 방법을 시도해보세요.
