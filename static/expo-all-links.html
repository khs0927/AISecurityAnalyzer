모든 가능한 Expo 연결 URL

아래는 HeartCare Expo 앱 연결을 위한 모든 가능한 URL 형식입니다. 여러 형식을 시도해보세요.

플랫폼 URL 형식 작업
iOS/Android exp://exp.host/@hsml/heart-care-expo?runtime-version=exposdk:50.0.0 열기
iOS (Safari) exps://exp.host/@hsml/heart-care-expo?runtime-version=exposdk:50.0.0 열기
Android/iOS expo://exp.host/@hsml/heart-care-expo?runtime-version=exposdk:50.0.0 열기
Android (인텐트) intent://exp.host/@hsml/heart-care-expo?runtime-version=exposdk:50.0.0#Intent;scheme=exp;package=host.exp.exponent;end 열기
iOS (유니버설) https://exp.host/--/to-exp/exp%3A%2F%2Fexp.host%2F%40hsml%2Fheart-care-expo%3Fruntime-version%3Dexposdk%3A50.0.0 열기
웹 https://expo.dev/@hsml/heart-care-expo 열기
대체 URL exp://u.expo.dev/update/heart-care-expo?runtime-version=exposdk:50.0.0 열기

참고: 위 링크들을 시도해보고 귀하의 기기에서 가장 잘 작동하는 링크를 사용하세요. 일반적으로 iOS에서는 exps://, Android에서는 exp:// 스킴이 권장됩니다.

수동 입력용 URL

Expo Go 앱에서 직접 입력할 수 있는 URL:

exp://exp.host/@hsml/heart-care-expo?runtime-version=exposdk:50.0.0

입력 방법: Expo Go 앱 실행 → 오른쪽 하단의 프로필 탭 → 상단의 'URL 입력' 선택 → 위 URL 붙여넣기

Expo Go 연결 가이드

iOS 연결 가이드

  1. App Store에서 Expo Go 앱을 설치하세요.
  2. Safari 브라우저를 사용하는 것이 가장 좋습니다.
  3. exps:// 또는 exp:// 스킴으로 시작하는 링크를 클릭하세요.
  4. "Safari에서 앱 열기" 또는 "Expo Go에서 열기" 메시지가 표시되면 수락하세요.
  5. 또는 Expo Go 앱에서 직접 URL을 입력하세요.

Android 연결 가이드

  1. Google Play 스토어에서 Expo Go 앱을 설치하세요.
  2. Chrome 브라우저를 사용하는 것이 가장 좋습니다.
  3. exp://, expo:// 스킴으로 시작하는 링크 또는 인텐트 링크를 클릭하세요.
  4. "Expo Go에서 열기" 메시지가 표시되면 수락하세요.
  5. QR 코드 스캔이 더 간편할 수 있습니다. Expo Go 앱을 열고 QR 코드를 스캔하세요.

문제 해결: 링크가 작동하지 않는 경우, 다음 사항을 확인하세요:

  • Expo Go 앱이 최신 버전으로 설치되어 있는지 확인
  • 인터넷 연결 상태 확인
  • 다른 브라우저 사용 시도
  • 다른 연결 방법 시도
  • 기기를 재부팅하거나 Expo Go 앱 캐시 삭제
