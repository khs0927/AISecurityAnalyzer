iOS에서 HeartCare 앱 연결하기

1 Expo Go 앱 설치

먼저 App Store에서 'Expo Go' 앱을 검색하여 설치하세요.

2 Expo Go 앱 실행

Expo Go 앱을 실행한 후, 오른쪽 하단의 '프로필' 탭을 누릅니다.

그런 다음 상단의 'URL 입력' 버튼을 찾아 탭합니다.

3 HeartCare URL 입력

다음 URL을 복사하여 Expo Go 앱의 URL 입력 필드에 붙여넣기 하세요:

exp://exp.host/@hsml/heart-care-expo?runtime-version=exposdk:50.0.0

4 대체 방법: 직접 열기

위 방법이 작동하지 않는 경우, 아래 버튼을 탭하여 직접 앱을 열어보세요:

Expo Go로 직접 열기

참고: iOS에서는 Safari 브라우저를 사용하는 것이 가장 안정적입니다. Chrome이나 다른 브라우저를 사용하고 있다면 Safari로 전환하세요.

5 문제 해결

연결에 문제가 있는 경우:

  • Expo Go 앱이 최신 버전인지 확인하세요
  • Safari 브라우저를 사용하세요
  • 인터넷 연결이 안정적인지 확인하세요
  • iOS 설정 앱에서 Safari 설정을 확인하고 쿠키와 자바스크립트가 허용되어 있는지 확인하세요
  • Expo Go 앱을 완전히 종료하고 다시 시도하세요
연결 테스트 실행

Android에서 HeartCare 앱 연결하기

1 Expo Go 앱 설치

먼저 Google Play 스토어에서 'Expo Go' 앱을 검색하여 설치하세요.

2 QR 코드로 연결

Android에서는 QR 코드 스캔이 가장 간단한 방법입니다. 아래 QR 코드를 Expo Go 앱에서 스캔하세요:

더 큰 QR 코드 보기

3 수동 URL 입력

QR 코드 스캔이 작동하지 않는 경우, Expo Go 앱에서 'URL로 열기' 옵션을 선택하고 다음 URL을 입력하세요:

exp://exp.host/@hsml/heart-care-expo?runtime-version=exposdk:50.0.0

4 직접 열기

또는 아래 버튼을 탭하여 직접 앱을 열어보세요:

Expo Go로 직접 열기 대체 스킴으로 열기

5 문제 해결

연결에 문제가 있는 경우:

  • Expo Go 앱이 최신 버전인지 확인하세요
  • Chrome 브라우저를 사용해보세요
  • 다른 네트워크(WiFi, 모바일 데이터)를 시도해보세요
  • Expo Go 앱의 권한 설정을 확인하세요
  • 폰을 재부팅하고 다시 시도해보세요

연결 후 사용 방법

성공적으로 연결되면: HeartCare 앱이 Expo Go 내에서 실행됩니다. 이제 앱의 모든 기능을 사용할 수 있습니다.

주요 기능

  • 심장 진단: 실시간 ECG/PPG 데이터 모니터링
  • 응급 처치: 상황별 응급 처치 가이드 확인
  • 비상 연락: 긴급 상황 시 보호자에게 알림 전송
  • AI 상담: 건강 관련 질문에 답변 제공

앱 사용 중 문제가 발생하면 메인 화면에서 설정 아이콘을 탭하여 도움말을 확인하세요.

주의: Expo Go 앱을 종료하면 HeartCare 앱도 함께 종료됩니다. 다시 연결하려면 위 단계를 반복하세요.
