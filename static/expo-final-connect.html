알림: 위 링크를 클릭하면 Expo Go 앱이 자동으로 실행되고 HeartCare 앱이 로드됩니다.

방법 1: URL 직접 입력

  1. Expo Go 앱을 실행하세요
  2. "프로필" 탭으로 이동하세요
  3. "URL 입력"을 탭하세요
  4. 아래 URL을 복사하여 붙여넣기 하세요
exp://exp.host/@hsml/heart-care-expo?runtime-version=exposdk:50.0.0

방법 2: Safari로 열기

  1. Safari 브라우저에서 아래 URL을 입력하거나 클릭하세요
  2. "앱 열기" 확인 메시지가 나타나면 허용하세요

연결이 안 되나요?

다음 사항을 확인하세요:

  • Expo Go 앱이 최신 버전으로 설치되어 있는지 확인하세요
  • 인터넷 연결이 안정적인지 확인하세요
  • 다른 네트워크(모바일 데이터 등)로 시도해보세요
  • Safari 브라우저 대신 Chrome을 사용해보세요
  • 때로는 Expo 서버 자체 문제일 수 있습니다. 잠시 후 다시 시도하세요
